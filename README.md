Как запустить проект:
npm run dev

открыть в браузере http://localhost:3000


FSD-архитектура:
src -> app
components - папки с миссиями для каждой сцены, меню игры, модальные окна, экраны загрузки, игрок
objects - папки с 3D объектами для каждой сцены
scenes - сцены каждой из комнат
stores - хранилище состояния миссий
pages - папки каждой из станиц (комнат)
